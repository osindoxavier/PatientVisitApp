# PatientVisitApp

A simple Android app for registering patients, recording vitals, and conducting assessments based on BMI.

## Tech Stack
- Kotlin + Jetpack Compose
- Ktor (Networking)
- Room (Local DB)
- MVVM + Clean Architecture
- DataStore for tokens

## Setup
1. Clone the repository
2. Open in Android Studio
3. Run the app (minSdk 24)
